## [0.1.0] - 2025-01-17

### Added

- CLI with `lint` and `version` commands using cobra
- JSON Schema parser for linting (supports `$defs`, `definitions`, `anyOf`, `oneOf`)
- Union detection with discriminator field analysis
- Nullable pattern detection (`anyOf: [T, null]`)
- Reference pattern detection (`anyOf: [ComponentReference, BaseXxx]`)
- Configurable discriminator field priority (`component_type`, `type`, `kind`)
- Multiple output formats: text (default), JSON, GitHub Actions annotations
- Exit codes: 0 (ok), 1 (errors), 2 (warnings only)
